THE BEST PROGRAM TO GUIDE YOU TO YOUR NEXT CAREER

HTML 5

In the initial part of the Front-End Academy you will learn the basics of programming - HTML. You will start with the basics, you will continue towards advanced learning of this language. You will complete this segment with a task and a project.

CSS 3

In the initial part of the Front-End Academy you will learn the basics of programming - HTML. You will start with the basics, you will continue towards advanced learning of this language. You will complete this segment with a task and a project.

JAVASCRIPT

The third pillar of technology on the World Wide Web is JavaScript, a highly functional, dynamic, and event-driven programming language. These features are just a few of the reasons why there is a great demand for developers with JavaScript skills.

REACT

The next module is the basic and advanced study of the React, open-source JavaScript library, designed to build a user interface. One of the reasons why there is a need for this library is the fact that through React you create web pages and mobile applications.

REACT + REDUX

Key part of completing Front-End Academy is the Redux - an open-source JavaScript library, designed to manage application status. Upon completion of this module, you’ll be able to create web page in a form of a blog.

FINAL PROJECT

The last module of the Front-End Academy will be focused on the final project - creating a React application.
